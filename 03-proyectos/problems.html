Recursos Generales Gratuitos

Plataformas y Entornos

  • ➔ Google Colab: Notebooks Jupyter gratuitos con acceso a GPU/TPU.
  • ➔ Kaggle: Plataforma con datasets, notebooks y competiciones.
  • ➔ Jupyter Notebook/Lab: Para desarrollo local e interactivo.

Frameworks y Librerías

  • ➔ Scikit-learn: Esencial para ML clásico (casi todo en esta lista).
  • ➔ Pandas: Para manipulación y análisis de datos.
  • ➔ TensorFlow/Keras: Para proyectos de Deep Learning (CV, NLP).
  • ➔ OpenCV: Librería clásica para tareas de Visión por Computadora.
  • ➔ Statsmodels/Prophet: Para Series de Tiempo (ARIMA, FBProphet).

Fuentes de Datasets

  • ➔ UCI Machine Learning Repository: Datasets clásicos para benchmarking.
  • ➔ Kaggle Datasets: Miles de datasets públicos y variados.
  • ➔ Google Dataset Search: Un motor de búsqueda para datasets.
  • ➔ Fuentes gubernamentales abiertas: Datos socioeconómicos, demográficos y financieros.
